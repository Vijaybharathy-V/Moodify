# Moodify:Emotion-Based Music Player

A desktop application that detects facial emotions in real time using a webcam and plays music corresponding to the detected emotion.

---

## Overview

The application uses computer vision and deep learning to analyze facial expressions and automatically select music that matches the userâ€™s emotional state. A simple graphical interface allows camera control, emotion monitoring, and music playback.

---

## Features

- Real-time webcam feed  
- Facial emotion recognition using DeepFace  
- Automatic emotion-based music playback  
- Manual emotion override  
- Adjustable analysis interval  
- Basic music controls (play, pause, stop)

---

## Technology Stack

- Python  
- Tkinter (GUI)  
- OpenCV (video capture and face detection)  
- DeepFace (emotion analysis)  
- Pygame (audio playback)  
- Pillow (image processing)

---

## Notes

- Good lighting improves emotion detection accuracy.
- A webcam is required.
- If emotion detection fails, the system defaults to a neutral state.
